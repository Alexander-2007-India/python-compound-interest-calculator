
# Python Compound Interest Calculator

A simple Python program that calculates compound interest and validates user input.

## Features

* Input validation for principal, rate, and time
* Uses the compound interest formula
* Displays the final balance in Indian Rupees (₹)

## Formula

A = P(1 + r/100)^t

Where:

* P = Principal amount
* r = Interest rate
* t = Time in years
* A = Final amount

## Example

Enter the principal amount: 10000

Enter the rate of interest: 10

Enter the time in years: 2

The total balance after 2 year/s is ₹12100.00

## How to Run

python compound_interest.py
